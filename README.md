📘 HOUEFA-DIGITAL — Encyclopédie HTML

Une encyclopédie interactive des balises HTML5 conçue pour apprendre, réviser et maîtriser le HTML de manière simple, moderne et efficace.

🚀 Aperçu

Ce projet propose une référence complète des balises HTML avec :

_ Plus de 100 balises HTML5

_ Recherche en temps réel

_ Filtres par type (bloc, inline, métadonnées, etc.)

_ Descriptions détaillées pour chaque balise

_ Interface moderne, fluide et responsive

_ Démo

👉https://github.com/jenniferabh8-cell/html-encyclopedie

 - Objectif

Ce projet a été créé pour :

.Aider les débutants à comprendre le HTML

.Offrir une référence rapide aux développeurs

.Servir de support pédagogique pour ebooks ou formations

.Centraliser toutes les balises HTML en un seul endroit
